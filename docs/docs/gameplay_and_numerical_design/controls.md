# 玩家基础战斗键位

## 键盘和鼠标

```
Esc - 游戏菜单
WASD - 移动
Space - 跳跃
Shift - 疾跑
Ctrl - 翻滚
Alt - 下蹲
Q - 使用选中的物品栏物品
E - 时空边界手的技能
1234 - 切换武器
R - 武器的技能
T - 弹反
鼠标左键 - 普通攻击
鼠标右键 - 重击
鼠标滚轮（滚动） - 切换物品栏选中
鼠标滚轮（按下） - 索敌
```

## 手柄

```
菜单 - 游戏菜单
L - 移动
Y - 跳跃
L3 - 疾跑
B - 翻滚
A - 下蹲
Up - 使用选中的物品栏物品
LT - 时空边界手的技能
Left & Right - 切换武器
X - 武器的技能
LB - 弹反
RB - 普通攻击
RT - 重击
Down - 切换物品栏选中
R3 - 索敌
```
